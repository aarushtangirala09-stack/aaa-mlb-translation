# Beyond Raw wRC+: Which Triple-A Hitting Skills Translate to MLB?

Reproducibility repository for a research paper analyzing which AAA offensive
statistics predict Major League Baseball hitting performance.

**Author:** Aarush Tangirala
**Paper:** `aaa-translation-paper.pdf` (in this repo)

---

## Summary

Using FanGraphs data on 486 Triple-A hitters from 2015–2024 who reached MLB
with at least 200 plate appearances, the paper:

1. Regresses career-PA-weighted MLB wRC+ on five standardized AAA predictors
   (BB%, K%, ISO, BABIP z-scored within season; age at MLB debut), restricted
   to pre-debut AAA seasons to eliminate temporal leakage.
2. Builds an "AAA Translation Score" from the resulting weights and compares
   it head-to-head against raw AAA wRC+ using paired 5-fold cross-validation.
3. Applies inverse probability weighting (IPW) to correct for survivorship
   into the 200+ MLB PA cohort.

Headline findings: AAA ISO (+5.48 per SD, p < 0.001) and K% (−3.54, p < 0.001)
are the strongest translators; BB% is modest; age at debut is small; BABIP is
non-predictive. The Translation Score ties raw AAA wRC+ on cross-validated
prediction, but a model combining the two beats the baseline at p = 0.005.

## Data availability

This repository does **not** include the raw source data. FanGraphs
(fangraphs.com) is the sole source and their data is proprietary. To
reproduce this analysis end-to-end you will need to:

1. Obtain a **FanGraphs Membership** (paid subscription, roughly $15/month).
2. Export the two leaderboards described in `EXPORT_INSTRUCTIONS.md`:
   - **AAA batting**, 2015–2024, Advanced/Dashboard view, split by season,
     no PA minimum. Approximately 9,370 player-seasons.
   - **MLB batting**, 2015–2025, Advanced/Dashboard view, split by season,
     no PA minimum. Approximately 15,521 player-seasons.
3. Save them to `data/raw/aaa/` and `data/raw/mlb/` respectively.

Once those two CSVs are in place, the four analysis scripts reproduce every
figure, table, and coefficient in the paper. Cohort-level derived data
(`matched_players.csv` and `matched_players_with_score.csv`) is not shipped
because those files still contain FanGraphs's per-player values; they are
regenerated by running the pipeline.

**Original data pull date:** June 2026.

## What is included

**Analysis scripts** (`scripts/`):

- `01_load_and_match.py` — cleans, matches players by FanGraphs ID, applies
  the pre-debut temporal filter, computes within-season z-scored predictors
  and age at MLB debut.
- `02_regression.py` — fits three OLS specifications (baseline, skill,
  kitchen sink) plus ridge/lasso robustness; computes VIFs; saves the
  headline coefficient plot.
- `03_translation_score.py` — builds the AAA Translation Score, runs the
  paired cross-validation head-to-head against raw AAA wRC+, reports paired
  t-tests.
- `04_survivorship.py` — fits the logistic selection model on all 5,245 AAA
  hitters; applies IPW to the outcome regression; runs three sensitivity
  imputations.
- `05_final_figures.py` — regenerates the five headline figures with
  publication styling.
- `probe_extended_score.py`, `probe_pcl.py` — Option B extension probes
  (prospect-status features and PCL park exposure). Both fail to beat the
  baseline; documented in §6 of the paper.
- `make_pdf.py` — builds `aaa-translation-paper.pdf` from `Paper_Draft.md`
  via headless Chrome.

**Output tables** (`data/processed/`) — pure regression outputs (no raw
FanGraphs values):

- `regression_coefficients.csv` — Table 2 (skill / baseline / kitchen sink).
- `step3_model_comparison.csv` — Table 3 (competitor CV metrics).
- `survivorship_corrected_coefs.csv` — Table 4 (OLS vs IPW).
- `sensitivity_coefficients.csv` — sensitivity-analysis coefficients.

**Figures** (`figures/`) — the five paper figures plus three diagnostic
figures used during development.

**Paper source and PDF:**

- `Paper_Draft.md` — the Markdown source.
- `aaa-translation-paper.pdf` — the rendered paper.

## Reproducing the results

Assuming Python 3.9+ and the two FanGraphs CSVs are in place:

```bash
python -m venv .venv
.venv/bin/pip install pandas numpy scikit-learn statsmodels scipy \
                     matplotlib seaborn markdown

.venv/bin/python scripts/01_load_and_match.py
.venv/bin/python scripts/02_regression.py
.venv/bin/python scripts/03_translation_score.py
.venv/bin/python scripts/04_survivorship.py
.venv/bin/python scripts/05_final_figures.py
```

Each script prints its output to the console and writes derived CSVs to
`data/processed/` and figures to `figures/`.
